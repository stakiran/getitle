## やりたいこと
Cosense のプロジェクトからページタイトル一覧を自由に素早くつくりたい。ランダムに 100 件並べたり、python の slice のように start,end を指定して取り出したりしたい。

以下のように1タイトル1行で並べればよい。

```
[ページタイトル1]
[ページタイトル2]
[ページタイトル3]
...
```

## 戦略
できるだけ素早く実行したいので、以下のようにする。

- raw/*.json として Cosense の Exported json を置いておき、これをパースする
- まずは exported json からタイトル名一覧をつくる（タイトルキャッシュ）
- 一覧化は、タイトルキャッシュに対して行う
- 一覧化の UI は様々考えられるが、いったん standalone html をつくる
